\# TFG – Gestión Semántica de Inventario Sostenible en PYMEs



Repositorio del Trabajo Final de Grado (UOC).



Este proyecto desarrolla un sistema de gestión semántica del inventario para PYMEs del sector alimentación,

integrando datos de stock, caducidad y criterios de sostenibilidad mediante tecnologías de la Web Semántica

(RDF, OWL, SPARQL) y un panel de visualización con Streamlit.
